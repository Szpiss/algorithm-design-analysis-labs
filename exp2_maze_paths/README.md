# 实验二 迷宫路径

## 本实验完成了什么

本实验使用回溯法求解迷宫问题，输出从指定入口 `s` 到出口 `t` 的所有可行路径，并在所有路径中选出一条最短路径。

## 如何实现

程序采用深度优先搜索配合回溯：

1. 从入口开始尝试向上、下、左、右四个方向扩展。
2. 使用 `visited` 数组避免同一条路径内重复进入同一格，保证输出的是简单路径。
3. 当到达终点时，将当前路径加入答案集合。
4. 回溯撤销当前步骤，继续尝试其他分支。

## 使用的算法

- 算法策略：回溯法
- 解空间模型：子集树/状态空间树
- 核心思想：深度优先枚举所有可行路径，并通过回溯恢复现场
- 时间复杂度：最坏情况下为指数级

## 文件说明

- `src/main.cpp`：主程序
- `samples/sample_input.txt`：测试输入

## 输入格式

```text
m n
maze matrix(0 empty, 1 blocked)
sx sy
tx ty
```

坐标采用从 `0` 开始的行列编号。

## 输出内容

- 所有可行路径
- 路径总数
- 一条最短路径及其长度
