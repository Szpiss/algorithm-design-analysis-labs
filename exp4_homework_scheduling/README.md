# 实验四 赶作业

## 本实验完成了什么

本实验实现了一个最小扣分作业调度程序。对于每组测试数据，程序根据每份作业的截止日期和扣分值，计算最优完成顺序下的最少扣分。

## 如何实现

程序使用贪心策略：

1. 按扣分从大到小排序，优先安排“不做损失更大”的作业。
2. 对每份作业，尽量安排到不晚于其截止日期的最靠后空闲时间。
3. 使用并查集维护当前还能使用的最近空闲时间段，从而快速查找可安排位置。
4. 若某份作业无法安排到截止日期之前，则其扣分计入总罚分。

## 使用的算法

- 算法策略：贪心法
- 选择原则：每一步优先保住高罚分作业
- 数据结构：并查集
- 时间复杂度：`O(n log n)`
