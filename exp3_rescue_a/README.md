# 实验三 在原始森林中解救A

## 本实验完成了什么

本实验使用队列式分支限界法判断 B 是否能够在原始森林中找到 A。程序按照题目给定输入格式读取多组测试数据，并输出 `Yes` 或 `No`。

## 如何实现

程序把状态定义为 `(x, y, dogBites)`：

1. `x, y` 表示当前位置。
2. `dogBites` 表示已经被野狗咬过几次。
3. 使用队列按层次扩展活结点。
4. 若遇到 `k`，该分支直接剪枝。
5. 若进入 `d` 后累计被咬次数达到 2，则该分支剪枝。
6. 对于已经访问过的 `(x, y, dogBites)` 状态不再重复扩展。

## 使用的算法

- 算法策略：分支限界法
- 具体实现：队列式分支限界
- 核心思想：广度优先扩展活结点，并用约束条件作为界进行剪枝
- 时间复杂度：`O(n^2)`，状态数量最多为 `2 * n * n`

## 输入格式

与指导书保持一致：

```text
t
n
grid row 1
grid row 2
...
```

其中：

- `p`：B 的起点
- `a`：A 的位置
- `r`：空道路
- `k`：金刚，不能进入
- `d`：野狗，累计被咬两次会死亡
