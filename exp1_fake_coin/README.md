# 实验一 查找假币问题

## 本实验完成了什么

本实验实现了一个基于分治思想的假币查找程序。程序在已知总硬币数、假币编号和假币轻重属性的测试条件下，模拟天平称重过程，输出每一次称重的左右托盘硬币编号、称重结果，并最终给出假币位置和类型。

## 如何实现

实现采用两阶段递归策略：

1. `searchUnknown`：
   在假币位置和轻重属性都未知时，把候选硬币划分为三部分并进行比较。
2. `searchPolarized`：
   当一次称重后已经知道“假币若在左边则一定偏重，若在右边则一定偏轻”时，继续递归缩小候选范围。
3. `searchKnown`：
   当候选集合中所有硬币的异常方向已经确定后，使用三分递归快速定位目标。

程序同时记录称重日志，用于实验报告中的“称重过程”展示。

## 使用的算法

- 算法策略：分治法
- 核心思想：每次称重尽量把问题划分为三种互斥结果，对应地缩小搜索空间
- 时间复杂度：约为 `O(log_3 n)`，与天平三种结果的判定方式相匹配

## 文件说明

- `src/main.cpp`：主程序
- `samples/sample_input.txt`：测试输入

## 输入格式

```text
t
n fakeIndex type
...
```

- `t`：测试组数
- `n`：硬币总数，`n > 3`
- `fakeIndex`：假币编号，范围 `0 ~ n-1`
- `type`：`H` 表示假币偏重，`L` 表示假币偏轻

## 输出内容

程序输出每一轮称重过程、称重次数、最终找到的假币编号及其类型，并附带信息论下界作对照。
